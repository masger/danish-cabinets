Hypothesis 1: The “Long Name, Short Tenure” Law

Joker: “A Prime Minister doesn’t have time to pronounce long names during a crisis. If your name has more than 15 characters, you get fired faster.”

Code 
dat[, name_len := nchar(name)]
# Test if name length correlates with tenure length
cor_test <- cor.test(dat$name_len, dat$tenure_years, method="pearson")
cat("Correlation between name length and tenure:", round(cor_test$estimate, 3), "\nP-value:", round(cor_test$p.value, 4))
Correlation between name length and tenure: -0.037 
P-value: 0.2103

Data Scientist Verdict: Null hypothesis reigns. Name length does not impact tenure.
